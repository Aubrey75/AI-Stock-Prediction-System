# AI Stock Prediction & Recommendation System

## Overview

This project is a machine learning-based stock prediction and recommendation system built with Python.

The system analyzes historical stock market data and predicts future price movements using technical indicators and Random Forest models.

## Features

- 1-day stock movement prediction
- 5-day stock movement prediction
- Probability-based forecasts
- RSI technical indicator
- Moving Average indicators (MA5, MA20)
- Volume change analysis
- Stock recommendation engine
- Automatic ticker correction
- Watchlist management
- Historical backtesting

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- yfinance
- Matplotlib

## Machine Learning Model

The project uses Random Forest Classifiers with engineered features including:

- Return
- MA5
- MA20
- MA Difference
- Volume Change
- RSI
